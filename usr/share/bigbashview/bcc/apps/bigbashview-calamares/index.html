BTRFS sem fsync

A partição Btrfs trás diversas melhorias para o sistema, por exemplo, compactação de dados e pontos de restauração.

Em algumas configurações de particionamento o Btrfs pode ser incompatível.

Recomendo utilizar Btrfs, mas se a instalação não funcionar corretamente experimente instalar utilizando EXT4.

BTRFS com fsync

A partição Btrfs trás diversas melhorias para o sistema, por exemplo, compactação de dados e pontos de restauração.

Essa opção também desativa as chamadas de fsync de todos os programas.

Isso torna o sistema mais responsivo, porém, caso falte energia, pode ser que os arquivos salvos nos últimos segundos não estejam realmente salvos, então, ao ligar novamente o computador as últimas modificações podem ter sido revertidas.

Geralmente essa é a melhor opção para instalar o BigLinux.

EXT

Este é o tipo de particionamento mais tradicional para sistemas Linux, porém, possui menos recursos que o particionamento Btrfs.
